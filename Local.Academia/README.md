# 🏛️ Local Academia

A directory of Sri Lankan universities, research labs, and student projects active in AI/ML, safety, fairness, NLP, and policy.

## Sub-sections

- [`Universities/`](./Universities) - Per-institution profiles (faculty, labs, courses).
- [`Research.Papers/`](./Research.Papers) - Curated bibliography, BibTeX file.
- [`Student.Projects/`](./Student.Projects) - Undergraduate / postgraduate showcase.
