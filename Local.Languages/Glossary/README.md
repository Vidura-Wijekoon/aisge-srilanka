## Add a term

Submit a PR adding a row, with a short note on alternatives considered. If a term is contested, leave a comment in the `Notes` column rather than enforcing one choice.

## Style notes

- We prefer Indic-script primary entries with Romanised forms only where ambiguity arises.
- Where a calque is the dominant usage, we include it but flag with *(calque)*.
- For technical neologisms, we prefer descriptive coinages over phonetic transliteration.
