# Election Integrity

AI's intersection with Sri Lankan election cycles - manipulation, integrity, and counter-measures.

## Background

Sri Lanka has a mature, but stressed, election infrastructure. Recent elections (2024 presidential and parliamentary) saw measurable AI-enabled misinformation, with social media pages "grown then sold" to campaigns - a documented monetisation pattern.

## Threat model (working draft)

1. **Synthetic media of candidates** - voice clones, manipulated video.
2. **Coordinated inauthentic behaviour** - sockpuppet networks, paid amplification.
3. **AI-translated disinformation** spreading across Sinhala, Tamil, English populations.
4. **Foreign interference** - diaspora-driven narratives, cross-border ad spend.
5. **Voter-suppression nudges** - fake polling locations, false dates.
