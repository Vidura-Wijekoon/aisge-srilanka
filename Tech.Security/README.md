# 🔧 Tech & Security

Open-source projects, model evaluations, security tooling, and datasets relevant to AI deployed in Sri Lanka.

## Sub-sections

- [`Projects/`](./Projects) - Code & tools authored or contributed to by the SL community.
- [`Evaluations/`](./Evaluations) - Benchmarks, red-team transcripts, eval methodologies (esp. Sinhala/Tamil).
- [`Datasets/`](./Datasets) - Curated dataset catalogue with provenance and licence.

## Priority gaps (we are looking for contributors)

1. **Sinhala/Tamil red-teaming corpus** - adversarial prompts probing local-language failure modes.
2. **Local-context safety classifier** - content classifier tuned for SL hate-speech, defamation, and political-manipulation taxonomies.
3. **Deepfake provenance toolkit** - pipeline for journalists to verify SL-context image/video.
4. **Sinhala/Tamil eval suite** - translation-quality, toxicity, factuality benchmarks.

## Security advisories

If you discover a security issue in any project linked here, please follow responsible disclosure to the upstream maintainer first, then notify the chapter.
